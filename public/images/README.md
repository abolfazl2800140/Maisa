# راهنمای تصاویر

## ساختار فولدرها

```
images/
├── products/          # تصاویر محصولات
│   ├── backpack-1-1.jpg
│   ├── backpack-1-2.jpg
│   ├── laptop-1-1.jpg
│   └── school-1-1.jpg
├── categories/        # تصاویر دسته‌بندی‌ها
│   ├── backpack.jpg
│   ├── laptop.jpg
│   └── school.jpg
├── blog/             # تصاویر بلاگ
│   ├── blog-1.jpg
│   └── blog-2.jpg
└── logo.svg          # لوگوی سایت
```

## سایز پیشنهادی تصاویر

- **محصولات**: 800x800px (مربع)
- **دسته‌بندی**: 1200x600px (افقی)
- **بلاگ**: 1200x630px (افقی)
- **لوگو**: SVG یا PNG با پس‌زمینه شفاف

## فرمت‌های پشتیبانی شده

- JPG/JPEG (برای عکس‌ها)
- PNG (برای تصاویر با پس‌زمینه شفاف)
- SVG (برای لوگو و آیکون‌ها)
- WebP (برای بهینه‌سازی بیشتر)

## نکات مهم

1. نام فایل‌ها باید با نام‌های موجود در `data/*.json` مطابقت داشته باشد
2. از تصاویر با کیفیت بالا استفاده کنید
3. قبل از آپلود، تصاویر را بهینه‌سازی کنید
4. از alt text مناسب برای SEO استفاده شود
