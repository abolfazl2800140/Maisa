# فروشگاه مایسا - Maysa Shop

فروشگاه آنلاین کیف و کوله پشتی با Next.js 14

## ویژگی‌ها

- ✅ Next.js 14 با App Router
- ✅ TypeScript
- ✅ Tailwind CSS
- ✅ Product Carousel با Swiper.js
- ✅ بهینه‌سازی کامل SEO
- ✅ Responsive Design
- ✅ Mock Data برای شبیه‌سازی Backend
- ✅ صفحات کامل فروشگاهی

## صفحات

- صفحه اصلی با Hero و Product Carousel
- فروشگاه با فیلتر و جستجو
- جزئیات محصول با گالری Carousel
- سبد خرید
- فرآیند چک‌اوت
- بلاگ
- پنل کاربری

## نصب و اجرا

```bash
# نصب وابستگی‌ها
npm install

# اجرای سرور توسعه
npm run dev

# بیلد برای پروداکشن
npm run build

# اجرای نسخه پروداکشن
npm start
```

پروژه روی [http://localhost:3000](http://localhost:3000) اجرا می‌شود.

## ساختار پروژه

```
maysa-shop/
├── app/                    # صفحات و روت‌ها
├── components/            # کامپوننت‌های React
│   ├── carousel/         # Product Carousel
│   ├── layout/           # Header, Footer
│   ├── product/          # ProductCard
│   └── ui/               # کامپوننت‌های UI
├── data/                  # Mock Data (JSON)
├── types/                 # TypeScript Types
├── public/               # فایل‌های استاتیک
└── lib/                  # توابع کمکی
```
